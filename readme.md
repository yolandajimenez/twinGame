# Twin Game

Ejemplos para practicar

### Instalación recomendada

- Descargar [Firefox Developer](https://www.mozilla.org/es-ES/firefox/developer/)
- Descargar [Node.js](https://nodejs.org/)

### Start

```sh
$ npm install
$ npm start
```
